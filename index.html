INFERENCE

Inference was created for small teams to get instantaneous feeedbacks. The concept is to start a session to rate on specific feedback items on a 1-5 scale. Each voter finishes and passes on to the next voter and finally when the session finishes, the results are the averaged on each item. The vote's are not registered in the system,it dies when the session is finished. Each feedback item is configurable and customizable. They are stored in the system for next time use. Themes for the feedback items are put together from the characters based on 'The Lego Movie'. This application utilizes opensource frameworks like simplegrid responsive layout , jQuery , Ruby on Rails etc.

This is a simplistic tool determining the team health and direction.
